STEPS TO INSTALL APP

 

  1. Download App to PCs/Laptop or Open this link in Mobile Browser.
  2. Copy the .apk file to your Phone memory or in case of Mobile Browser just click on download button and it will initiate installation.
  3. Open this .apk file in the phone.
  4. Follow the instruction to install app on phone.

 

 

Note: Some phones do not allow to install third party apps from unknown source. So you will require to enable a feature in "security settings" under "Device Administration" which says "unknown source". This will allow you to install "display board" app on your phone. After installing app on your phone you are adviced to please disable this function again.
